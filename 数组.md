# chap1.数组

## 0.数组理论基础
* 存储空间连续（高维也连续顺下来）
* 相同数据类型、下标from 0
* 数组元素不能删，只能覆盖

## 704. 二分查找
* 什么时候用二分查找：题目指出**有序、无重复**
* 注意明确区间定义：
    左闭右闭：target在[left, right]区间
    `while (left <= right)` 
    `if (nums[middle] > target) right = middle - 1`
    左闭右开：target在[left, right)区间
    `while (left < right)` 
    `if (nums[middle] > target) right = middle`
